## sumArray.java - Method

```java
// sumArray.java
public static double sumArray(double[] nums) {
  double sum = 0;
  for (double i : nums) sum += i;
  return sum;
}
```

**Method parameters**
<br>Array of doubles to sum &mdash; `nums` as `double[]`

**Return value**
<br>With parameters `new double[]{1.0,2.5,3.2,6.3}`
```
13.0
```
